**About the Talk:**
As we increasingly share information *on* the web, it can be useful to build interactive data visualizations in the language *of* the web (i.e., JavaScript). [Observable Plot](https://observablehq.com/@observablehq/plot), a JavaScript library built on the grammar of graphics, is a user-friendly tool to create and customize data visualizations, then add interactivity---fast. In this talk and demo, you can follow along to build interactive charts using Observable Plot. In particular, this workshop will highlight how your existing grammar of graphics knowledge (e.g. in ggplot2) maps onto Plot, and puts you on the fast track to data visualization in JavaScript.
